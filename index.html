HTML e a logo HTML5

HyperText Markup Language (HTML) é uma linguagem de marcação. Documentos que se utilizam dele (para posteriormente serem abertos em um navegador web) são costruídos a partir de elementos dos mais diversos tipos e para diferentes propósitos.

Elementos

Entre os elementos HTML existentes, podemos citar alguns dos mais comuns:

  • <h1> - <h6>, <p> e <a>: títulos (para nomear seções e hierarquias), parágrafos (para definir parágrafos) e links (para referenciar outros documentos), sequencialmente.
  • <img>, <audio> e <video>: usados para incorporar imagens, áudios e vídeos em um documento HTML, sequencialmente.
  • <table>: usado para representar dados em formato de tabela, contendo opções para implementar legendas e sumários.

Estrutura geral

A estrutura geral de um documento HTML é composta da seguinte maneira:

  1. Um doctype: são requeridos por razões históricas; devem ser utilizadas para que os navegadores renderizem as páginas de maneira correta. O doctype é composto pela string "<!DOCTYPE html>".
  2. O elemento <html>: o elemento html envolve todo o conteúdo da página. Quando aplicado o atributo lang neste elemento, o valor especificado é definido como a linguagem primária do elemento.
  3. O elemento <head>: este elemento é responsável por assegurar os metadados da sua página; elementos como <meta> e <title> são bons exemplos.
  4. O elemento <body>: Ao contrário do elemento head, o body contém todo o conteúdo visível da sua página (ou seja, que irá aparecer quando o usuário acessar sua página). Texto, imagens, áudios, vídeos ou mesmo jogos estão inclusos.

A logo

A logo "HTML5" (visível na seção de introdução da página) foi criada pela World Wide Web Consortium (W3C). A página oficial "W3C HTML5 Logo" está disponível online e permite o download da logo em diferentes cores e proporções; um "construtor de distintitivo" também é apresentado e inclui um total de oito opções de personalização (podendo ser posicionado de forma verical ou horizontal). O termo HTML5 é usado para se referir às tecnologias web modernas, algumas delas sendo desenvolvidas na WHATWG.

De acordo com a página da Wikipedia sobre o HTML5, a logo foi introduzida em 18 de janeiro de 2011 e se tornou oficial em 1 de abril do mesmo ano. Inicialmente, a logo foi anunciada como "uma entidade visual para um amplo conjunto de tecnologias web abertas", mas logo depois, sua descrição foi alterada para "representa o HTML5, a base para aplicativos Web modernos".
